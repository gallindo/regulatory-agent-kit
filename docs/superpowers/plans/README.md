# Superpowers Plans

This directory contains Qwen Code Superpowers plan files for automated documentation generation, code reviews, and other reusable workflows.

## Usage

Plans stored here can be invoked via the Qwen Code CLI to run standardized workflows across the project.

## Contributing

When adding a new plan:
1. Name the file descriptively: `plan-name.yaml`
2. Include a brief description at the top of the file
3. Reference the plan in the project's `CLAUDE.md` or `README.md` if it should be discoverable by contributors
